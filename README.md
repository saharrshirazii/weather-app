## âš™ï¸ How It Works
The app uses the **OpenWeatherMap API** to fetch weather data for a given city.
When the user enters a city name and clicks the search button:
1. The app makes an API call using fetch().
2. It retrieves weather details (temperature, humidity, wind speed, etc.).
3. The UI updates dynamically with the fetched data.
4. If the city name is invalid, an error message is shown.
